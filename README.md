# Method-pro Technodom case 2
## Проект, выполняемый командой #11

### Для установки установки сервера на ваше устройство, следуйте следующим инструкциям:

* Установить **PostgreSQL**
* Запустить сервер базы данных, используя команду *pg_ctl -D "C:\Program Files\PostgreSQL\11\data" start*
* Создать новую базу данных, используя **pgAdmin4**
---------------------------------
* Скачать репозиторий в **.zip** формате и распаковать его
* В командной строке перейти в директорию с проектом
* Установить собственную локальную среду для проекта, прописав команду *virtualenv env*
* Активировать локальную среду командой *env\Scripts\activate.bat*
* Установить необходимые пакеты командой *pip install -r requiremets.txt*
----------------------------------
* Привязать базу данных, прописав в файл **prod_settings** следующий код:
```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql_psycopg2',
        'NAME': '',
        'USER': '',
        'PASSWORD': '',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```
* Поля **Name**, **User** и **Password** заполнять в соответствии со значениями, указанными при создании базы данных
----------------------------------
* В коммандной строке перейти во вложенную директорию **server**
* Произвести необходимые миграции, выполнив команды *python manage.py makemigrations* и *python manage.py migrate*
* Запустить сервер командой *python manage.py runserver*
